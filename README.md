# sh5b6 Pages Payload

Published payload for GitHub Pages.

Expected URLs after GitHub Pages is configured to serve branch `sh5b6` from `/`:

- `/sh5b6/`
- `/sh5b6/codex_version/`

Contents:

- `index.html`: primary landing page
- `codex_version/index.html`: named alias route
- `.nojekyll`: disable Jekyll processing
